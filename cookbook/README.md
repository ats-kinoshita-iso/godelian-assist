# Cookbook

Golden baseline configurations for Claude Code projects. Copy these into your own repos as
starting points.

## Sections

### `claude-md/`

CLAUDE.md templates for different project types (Python, TypeScript, full-stack, etc.). Each
template includes project structure, code style rules, and common commands.

### `hooks/`

Reusable hook recipes for `.claude/settings.json`. Examples: auto-lint on file write,
registry regeneration on stop, commit message validation.

### `mcp/`

MCP server configurations for common integrations (GitHub, databases, APIs). Drop these into
your project's `.mcp.json`.

## Usage

Browse the section you need and copy the relevant files into your project. These are starting
points — customize them for your specific setup.
