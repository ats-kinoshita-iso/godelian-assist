# CLAUDE.md Templates

Project instruction templates for different project types. Copy and customize for your repo.

## Planned Templates

- `python.md` — Python project with uv, ruff, mypy, pytest
- `typescript.md` — TypeScript project with bun/npm, biome, tsc
- `fullstack.md` — Full-stack project with both Python and TypeScript
- `minimal.md` — Bare minimum CLAUDE.md for any project
