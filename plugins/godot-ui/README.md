# godot-ui

UI implementation skills for Godot 4.x: HUD, menus, themes, and signal-driven data flow.

## Skills

| Skill | Invocation | Description |
|---|---|---|
| hud-design | `/hud-design` | CanvasLayer HUD with TextureProgressBar and EventBus wiring |
| menu-design | `/menu-design` | Pause/main menus with get_tree().paused and FocusMode navigation |
| theme-system | `/theme-system` | Project-wide Theme resource, StyleBoxFlat, font hierarchy |
| ui-signals | `/ui-signals` | EventBus signal flow between gameplay and UI |
