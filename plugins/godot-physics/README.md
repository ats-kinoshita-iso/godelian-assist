# godot-physics

Physics implementation skills for Godot 4.x: character movement, collision layers, detection zones, rigidbodies, and projectiles.

## Skills

| Skill | Invocation | Description |
|---|---|---|
| character-movement | `/character-movement` | CharacterBody3D with coyote time, jump buffering, air acceleration |
| collision-layers | `/collision-layers` | CollisionLayer constants class and layer/mask setup |
| area-detection | `/area-detection` | Area3D detection zones with ghost frame mitigation |
| rigidbody-patterns | `/rigidbody-patterns` | RigidBody3D force application, freeze modes, ragdoll switching |
| projectile-system | `/projectile-system` | Pooled projectile system for bullets and thrown objects |
