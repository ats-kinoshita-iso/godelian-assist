# game-design

Game design skills for 3D action-RPGs in Godot 4.x: systems design, core loop analysis, and balance tuning.

## Skills

| Skill | Trigger | Description |
|-------|---------|-------------|
| `systems-design` | "design the X system", "spec out combat/inventory" | Full system spec: data model, state machine, Godot skeleton |
| `game-loop` | "design the core loop", "how does the player progress?" | Three-level loop analysis (micro/meso/macro) with improvement proposals |
| `balance-design` | "balance the combat", "tune the numbers" | Numeric targets, progression curves, Resource schema for designer tuning |

## Design Philosophy

This plugin bridges game design intent to Godot implementation. Every skill produces:
1. A design rationale with concrete numbers or state machines
2. A Godot-specific implementation sketch (node structure, Resource schema, GDScript skeleton)
3. Edge cases and validation scenarios
