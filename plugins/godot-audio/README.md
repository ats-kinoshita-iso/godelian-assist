# godot-audio

Audio implementation skills for Godot 4.x: bus architecture, adaptive music, and SFX pooling.

## Skills

| Skill | Invocation | Description |
|---|---|---|
| audio-architecture | `/audio-architecture` | Bus structure, volume control, bus effects |
| adaptive-music | `/adaptive-music` | State machine + crossfade transitions between music stems |
| sfx-patterns | `/sfx-patterns` | AudioCue resources, SFX pool, pitch randomization, 3D audio |
